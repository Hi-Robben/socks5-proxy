需要開啓本地socks主機
終端代理需要添加規則
在~/.bashrc文件中增加以下两句，表示bash终端中的http和https的请求也通过socks5协议进行代理转发。
export http_proxy="socks5://127.0.0.1:2553"
export https_proxy="socks5://127.0.0.1:2553"
